DMARKET.KG — Единый SEO-файл (безопасно для админки)

Задача: добавить SEO-настройки, не ломая связку с админкой и не меняя дизайн. Варианты внедрения ниже. Рекомендуемый — Вариант B (append-only).

Куда вставлять

  1. Откройте шаблон, где находится тег <head>.
  2. Вариант A (замена): замените содержимое между <head>...</head> на блок ниже «Вариант A».
    Подходит, если админка разрешает редактировать весь head.
  3. Вариант B (append-only): если замена рискованна, просто вставьте блок ниже «Вариант B» сразу перед строкой </head>. Ничего не удаляйте. Это безопасно для админки.

Важно

  • Все теги в пакетах — невидимы для пользователей (meta/link/script JSON-LD).
  • Не трогает CSS/JS и верстку. Дизайн не меняется.
  • Маркет круглосуточный — в JSON‑LD указан режим 00:00–23:59 для всех дней.

Рекомендации (необязательные, можно позже)

  • Проверьте, чтобы не было 2+ одинаковых мета-тегов description, robots, og:image. Дубли можно оставить на первое время, но лучше чистить постепенно.
  • Убедитесь, что twitter:card имеет значение summary_large_image (не URL).
  • Используйте одну актуальную картинку 1200×630 для og:image и twitter:image.

Дополнительно (опционально, для корня сайта)

Если их нет — добавьте в корень сайта:

robots.txt

User-agent: *
Allow: /
Sitemap: https://dmarket.kg/sitemap.xml

sitemap.xml (пример каркаса)

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="https://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://dmarket.kg/</loc>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <!-- категории и товары добавляйте генератором -->
</urlset>
